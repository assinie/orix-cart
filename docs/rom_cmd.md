# ROM: rom_cmd

### ROM bank example

## DESCRIPTION
**rom_cmd.rom** add 3 commands to Orix shell.

## COMMANDS
*  hello
                display "Hello world!"
*  bonjour
                display "Bonjour le monde!"
*  bankid
                display the bank id of rom_cmd
